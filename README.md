# Corporate SaaS Subscription Manager

A simple, menu-driven Java console application to track and manage software tools across different company departments. This project demonstrates core concepts of **Java Database Connectivity (JDBC)** and SQL database operations.

## 🚀 Features
- **Create:** Add new software assets with costs, active seats, and renewal dates.
- **Read:** Display software lists cleanly using relational SQL `JOIN` statements.
- **Update:** Modify active seat counts, monthly pricing structures, or tool lifecycles.
- **Delete:** Remove outdated or canceled tools from the tracking database records.

## 🛠️ Tech Stack
- **Language:** Java (JDK 17 or higher)
- **Database:** MySQL Server
- **Driver:** MySQL Connector/J
- **API:** JDBC Core Framework
